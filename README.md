Sistema de Facturación - Arquitectura de Tres Capas

Este proyecto consiste en el desarrollo de un sistema de facturación implementado en C# utilizando el patrón de arquitectura de tres capas.
El sistema fue desarrollado con Windows Forms en Visual Studio y utiliza SQL Server para la gestión de la base de datos.

Tecnologías utilizadas

C#

Visual Studio

.NET Framework

Windows Forms

SQL Server

Arquitectura del proyecto

El sistema está organizado bajo el modelo de arquitectura de tres capas:

Capa de Presentación

Contiene los formularios que permiten la interacción entre el usuario y el sistema.

Capa de Negocio

Contiene la lógica del sistema y las reglas de negocio que controlan el funcionamiento de la aplicación.

Capa de Datos

Se encarga de la conexión con la base de datos y la ejecución de consultas SQL.

Funcionalidades del sistema

Sistema de inicio de sesión con validación de usuarios

Gestión de clientes

Gestión de productos

Creación y consulta de facturas

Administración de datos almacenados en la base de datos

Seguridad

El sistema cuenta con un módulo de autenticación que valida el acceso de los usuarios mediante credenciales almacenadas en la base de datos.
